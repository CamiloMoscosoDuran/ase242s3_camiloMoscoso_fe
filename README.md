# ¿Qué es Angular?
Angular es el framework de Google que utilizo para desarrollar aplicaciones web modernas y potentes de una sola página (SPA). Básicamente, es una plataforma "todo en uno" que ya trae las herramientas necesarias para manejar rutas, formularios y datos sin depender de mil librerías externas.

# ¿Qué herramientas necesitan instalar?
Para trabajar con él, lo primero que instalé fue Node.js, ya que necesito el gestor de paquetes npm para manejar las dependencias. Después, instalé el Angular CLI con el comando npm install -g @angular/cli, que es la herramienta de consola que uso para generar y correr mis proyectos rápidamente.
